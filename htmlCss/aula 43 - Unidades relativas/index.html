Unidades de Medida em CSS

  1. Existem duas categorias principais de unidades de medida em CSS: absolutas e relativas. As absolutas têm um tamanho fixo, independentemente do dispositivo, enquanto as relativas dependem de outras propriedades para definir tamanhos.
  2. Unidades de medida absolutas: px, pt, cm e mm.
  3. Unidades de medida relativas: %, em, rem, vh e vw.
  4. Usar medidas absolutas pode limitar a fluidez dos layouts na transição entre dispositivos, enquanto medidas relativas permitem uma maior flexibilidade.
  5. É possível definir valores mínimos e máximos para altura e largura usando as propriedades min-height, max-height, min-width e max-width, o que pode ser útil para limitar a expansão ou diminuição de elementos em diferentes tamanhos de tela.
